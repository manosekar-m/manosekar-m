#### 📈 [Stock Fuzzy Logic](https://github.com/manosekar-m/stock-fuzzy-logic) — AI Stock Price Prediction System

A **Fuzzy Logic-based** stock prediction engine using **RSI, MACD and Daily % Change** as soft computing indicators. Built with Flask + Python + scikit-fuzzy to classify market trends (Up / Down / Neutral) and forecast next-day prices. Trained on 40 years of Apple stock data, deployed on **Render**.
